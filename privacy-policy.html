Politique de Confidentialité
Auto Refresh Random Toggle

Dernière mise à jour : 13 février 2026

Cette politique de confidentialité explique comment l’extension Auto Refresh Random Toggle (versions Freemium et Premium) traite les données des utilisateurs. Nous accordons une importance essentielle à la confidentialité. Toutes les données sont traitées localement, directement dans votre navigateur.

1. Données collectées

1.1 Version Freemium

La version Freemium ne collecte aucune donnée personnelle.

Les seules données enregistrées localement sont :

  • Les URLs que vous choisissez de rafraîchir
  • L’intervalle de rafraîchissement
  • L’historique local (limité à 5 entrées)

Ces données restent dans votre navigateur et ne sont jamais envoyées à un serveur.

1.2 Version Premium

La version Premium enregistre localement :

  • Les URLs suivies
  • Les paramètres de rafraîchissement
  • L’historique complet (jusqu’à 100 entrées)
  • L’état de la licence Premium (clé ou token)

Ces données restent locales, ne sont jamais vendues, partagées ou transférées, et ne contiennent aucune information personnelle identifiable.

2. Données non collectées

L’extension ne collecte PAS :

  • nom, prénom
  • adresse e-mail
  • adresse IP
  • localisation GPS
  • informations financières
  • communications personnelles
  • historique complet de navigation
  • données de santé
  • données sensibles

3. Permissions utilisées

tabs

Permet d’identifier les onglets sélectionnés et de déclencher leur rafraîchissement.

alarms

Permet de programmer les intervalles de rafraîchissement (fixes ou dynamiques).

storage

Permet d’enregistrer vos préférences localement (URLs, intervalle, historique, licence Premium).

Aucune permission n’est utilisée pour collecter ou transmettre des données personnelles.

4. Code distant

L’extension n’utilise pas de code distant. Aucun script externe, CDN ou ressource dynamique n’est chargé depuis un serveur tiers.

Si une future version Premium nécessite une vérification de licence, celle‑ci se fera via une requête sécurisée, sans transmission de données personnelles.

5. Partage et transfert de données

Nous ne :

  • vendons pas les données
  • transférons pas les données
  • utilisons pas les données à des fins publicitaires
  • analysons pas votre navigation

6. Sécurité

L’extension fonctionne entièrement dans l’environnement sécurisé de Chrome. Aucune donnée personnelle n’est stockée ou transmise.

7. Suppression des données

Vous pouvez supprimer toutes les données locales en désinstallant l’extension ou en réinitialisant les paramètres via l’interface.
